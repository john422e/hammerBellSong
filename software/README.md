# hammer, bell, song

**Version:** 1.2
**Author:** John Eagle

---

## Overview

*hammer, bell, song* is a composition for seven instrumentalists (cello + six sustaining instruments) with electronics designed to run on iPhone using the MobMuPlat app. The phones are placed within a 3d printed "cricket" available from the composer or printed from files included here.
Each performer except the cellist operates an individual device as part of a coordinated ensemble.

---

## Platform

* ✅ iPhone (iOS) supported
* ❌ Android currently not supported

---

## Requirements

* iPhone
* MobMuPlat app:
  https://apps.apple.com/us/app/mobmuplat/id597679399

---

## Installation

1. Install **MobMuPlat** from the App Store.
2. Transfer the following files to your iPhone:

   * `HBSgui1.2.mmp`
   * `HBSsynth_mmp_1.2.pd`
3. Place both files inside the **MobMuPlat** directory on your device.

---

## App Setup

1. Open **MobMuPlat**.
2. Tap the **three-line menu icon** (upper left corner).
3. Select the **Audio/MIDI** tab.
4. Enable the following options:

   * **App Runs in Background**
   * **Override Audio Input**
5. Navigate to the **Select Document** tab.
6. Open:
   `HBSgui1.2.mmp`

---

## Performance Instructions

### Initialization

* Each of the six ensemble members uses one phone.
* Using the central knob:

  * Select a **unique position** (no duplicates).
  * The performer assigned to the *first position* must still press the knob once to initialize data.

---

### Starting the Piece

* Partially assemble the crickets (middle part screwed into bottom hemisphere with phone mount installed and phone secure in mount)
* When cued by the cellist (on stopwatch):

  * All performers press the **blue square button simultaneously**.
  * Screw on the top hemisphere and place cricket on floor in front of or next to you.
  * Cellist starts playing after 30 seconds have passed, at which point the rest of the ensemble may begin playing.

---

## Volume Calibration

* The **volume slider (right side)** is for testing only.
* Before performance:

  * All phones should be manually set to **equal volume levels** using the phone's volume settings (not the app). 

---

## Notes

* Ensure all devices are properly configured before the performance begins.
* Consistent volume across devices is essential for balance within the ensemble.
